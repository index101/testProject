# Deploy to Vercel (Free)

## Quick deploy

1. Push this project to GitHub
2. Go to [vercel.com](https://vercel.com) → Sign in with GitHub
3. Click **Add New** → **Project**
4. Import your repo
5. **Root Directory:** Set to `web` (important!)
6. Click **Deploy**

## Root directory

The Next.js app lives in the `web` folder. In Vercel project settings:

- **Root Directory:** `web`

## Local run

```bash
cd web
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
